# Weather Station Temperatures

A React application that displays temperature data from two Weather Underground personal weather stations.

## Features

- Displays current temperatures from two sensors (KMAWEBST38 and KMAWEBST37)
- Shows a historical temperature chart for the last 3 days
- Responsive design that works on desktop and mobile

## Development

```bash
npm install
npm run dev
```

## Deployment

The application is configured to deploy to GitHub Pages:

```bash
npm run deploy
```

This will build the application and publish it to the `gh-pages` branch.
